# Design Guide — Better Designer Club

> 이 문서는 이 프로젝트의 디자인 기준 예시이다.
> 색감 등을 참조하라는 것이 아닌 지시 사항을 아래 내용처럼 준수한다는 가이드 지표이다.
> 브랜드 명 등 역시 가명으로 작성된 문서이기에 현재 구축되어 있는 코드베이스에서 수정할 필요가 없다. (이미 브랜드명 등은 잡혀 있음)
> 규칙과 어긋나면 만들기 전에 멈추고 나에게 물어본다.

## 1. Overview (브랜드 한 줄·무드)

- 브랜드명: Better Designer Club (BDC)
- 한 문장: AI 시대, 보다 나은 디자이너를 위한 클럽
- 무드 키워드: 절제된 / 타입 중심 / 따뜻한 포인트
- 느낌 비교: labs.google 같은 깔끔한 에디토리얼 O · 알록달록 사탕색 X

## 2. Colors (컬러)

- Primary (포인트): #FF5E00
- Secondary (서브 포인트): #FFE477
- Ink (제목·본문): #15171C
- Muted (보조 텍스트): #6B7280
- Surface (카드·띄): #F7F7F5
- Background (배경): #FFFFFF
- 규칙: 색은 위계에만. 포인트는 아이콘·브랜드명·버튼에만, 카드는 중립면.

## 3. Typography (타이포그래피)

- 폰트: Pretendard (전부 이 하나만)
- H1 64px / 800 · H2 32px / 800 · H3 18px / 700 · 본문 16px / Regular
- 라벨 14px / 700 (#FF5E00) · 행간 본문 1.6
- 규칙: 폰트는 하나. 위계는 크기·굵기로만.

## 4. Layout (간격·여백)

- 기준 단위: 8px 그리드 (8 / 16 / 24 / 32 / 48 / 64 / 96)
- 콘텐츠 최대 너비 1080px · 섹션 패딩 96px · 카드 간격 24px · 좌우 여백 24px
- 규칙: 위 단위 밖 어중간한 값 금지.

## 5. Elevation & Depth (그림자·깊이)

- 기본은 그림자 없음 — 면과 여백으로 구분한다.
- 호버 시에만 얙게: 0 16px 38px rgba(20,20,30,.08) + 살짝 떠오르기.
- 규칙: 라이트그레이 구분선으로 영역을 나누지 않는다.

## 6. Shapes (모서리·형태)

- 모서리: 카드 22px / CTA 패널 28px / 버튼 pill(완전 둥글게)
- 아이콘: 라인(블랙 스트로크) + 오렌지 도트
- 규칙: 둥글기 값은 위 세 가지로만 통일.

## 7. Components (컴포넌트)

- Primary 버튼: 배경 #FF5E00, 흰 글자, pill, 높이 50px
- Ghost 버튼: 흰 배경, 1px #E4E4E8 보더, pill
- 카드: 배경 #F7F7F5, 모서리 22px, 호버 시 살짝 떠오름
- 통계: 큰 숫자 44px / 800 + 보조 라벨

## 8. Do's and Don'ts (지킬 것·하지 말 것)

- 폰트 2개 이상 혼용 금지
- 카드 알록달록 채색 금지 (중립면 + 아이콘만 컬러)
- 그라데이션·과한 그림자·라이트그레이 선 금지
- 박스 좌측 컬러 스트라이프 금지
- 모션은 페이드업·카운트업 정도로 절제
